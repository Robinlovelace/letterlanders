# Session Log: 2025-12-05

## Initial Request
The user requested an educational app generalized from "Digits" (https://github.com/foxtrotdev/learn-digits).
**Goal:** A game called "LetterLanders" to teach letters/numbers.
**Theme:** Landing letters safely on the moon.
**Tech Stack:** Rust, TypeScript, Tauri. Support for Linux Terminal (TUI) and Mobile/Web.

## Architectural Decisions
1.  **Stack:** Rust (Core/TUI) + Tauri v2 (App) + Svelte (Frontend).
2.  **Input Method:** Direct keyboard input for TUI (vs arrow selection).
3.  **Audio:** Real audio playback in TUI using `rodio`.
4.  **Progression:** Session-based levels (variants for Letters/Numbers).

## Implementation Steps Completed
1.  **Workspace Setup:** Initialized Rust workspace with `core` and `tui` members.
2.  **Core Logic:** Created `GameEngine`, `SessionState`, and `GameVariant` in `core/src`. Implemented RNG level generation.
3.  **TUI:** Built a `ratatui` interface in `tui/src`. Added `rodio` for audio playback.
    *   *Note:* Required `libasound2-dev` on host system.
4.  **Tauri App:** Scaffolder `app` using `create-tauri-app` (Svelte + TS).
    *   Integrated `letterlanders-core` into `app/src-tauri`.
    *   Exposed `start_new_game` command to frontend.

## Current Status
*   TUI compiles and runs (`cargo run -p letterlanders-tui`).
*   Tauri backend compiles (`cargo check`).
